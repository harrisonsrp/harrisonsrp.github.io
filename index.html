About Me

I am a software engineer with over 2 years of experience, experienced in machine learning, web development, and the seamless integration of these fields to create innovative solutions. My expertise spans both relational and NoSQL databases, enabling efficient data storage and collection, and the development of robust machine learning models utilizing various optimization techniques and error analysis. I have extensive hands-on experience with generative AI. I specialize in fine-tuning LLMs using LoRA and PEFT methods as well as retrieval augmented generation (RAG), and integrating them with server-side applications like Flask. My background includes nearly two years as a web developer and an additional 1 year as a machine learning engineer, with a strong focus on deploying machine learning solutions in production environments. My technical skills encompass Python, SQL, GIT, Docker, and a range of machine learning libraries such as TensorFlow, Transformers and sklearn. I am proficient in MLOps platforms such as AWS SageMaker, S3 and Athena, leveraging these tools to streamline the ML lifecycle from data preparation to deployment and monitoring. My coding proficiency follows Object-Oriented Programming (OOP) principles.

I actively engaged in online courses, acquire certifications, participate in workshops and grow my network in the area of data analysis to remain up-to-date with the latest advancements in the field.
